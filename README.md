# VideoRecording
p2p 录制视频需求

根据p2p网贷平台 需求 所做的的视频录制 喜欢的欢迎使用

一、为什么要做短视频

- 大佬们本来自己在网上接了一个第三方sdk，功能很齐全，但不开发API，UI界面都是死的，由他们改。
- 我们需要的功能比较简单，就是录制视频，重录，播放，上传。
- 跟他们谈 API 价格的高于预计 
- 因此：
    我们决定自己做，而且网上有很多借鉴

二、先来几张效果图。
      1、30秒短视频 可以自定义
      2、开始3秒后 可以点完成或者重拍 不能停止（网贷需求）
      3、按钮外围进度和倒计时
